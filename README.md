# CS2-ESP-Cheat
### Using
1. Go to [pyMeow](https://github.com/qb-0/pyMeow/releases/tag/1.73.42).
2. Download pyMeow-1.73.42.zip
3. Command `pip install pyMeow-1.73.42.zip`
4. Run after opening the game!!!
